## Key Design Decisions Made

| Decision | Rationale |
|----------|-----------|
| INT score does NOT gate AI behavior | T-Rex (INT 2) attacks freely; only explicit lore creates passive creatures |
| `'defend'` AI profile | Explicit opt-in at spawn for "defends unless commanded" creatures (Giant Fly) |
| Summon-type = no numeric CR | Avatar of Death, Giant Fly correctly filtered from main bestiary |
| Chebyshev 3D distance | PHB default; DMG optional 5/10 diagonal explicitly excluded |
| Multiattack never OA | SAC v2.7 confirmed; `selectOAAction()` filters it out |
| Unarmed strike is fallback #6 | After all stat-block actions fail; before Dash |
| Battlefield bounds clamping | `executeMove()` clamps dest to grid — prevents infinite kite loops |
| Larva spawned as `attackNearest` | CR 0 simple fiend should not tactically self-preserve |
